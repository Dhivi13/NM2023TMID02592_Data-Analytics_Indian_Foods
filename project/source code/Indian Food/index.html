About Us

Learn More About Us

The project is based on exploratory data analysis on data set 'Indian Food Eda'. The data set contains the information about famous Indian dishes and the foodie. We plot some graphs to know the trends of food in different region and correlation between cooking time and food item, number of food items for each state veg-nonveg and recommendation of similar dishes using K-nearest neighbor and also collecting the dataset about the foodie.
